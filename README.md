# Installations
```sh
pip install -r requirements.txt
```

# Serving on localhost
```sh
python -m mkdocs serve
```
After running the command above, the thesis is available at [here](http://127.0.0.1:8000/).

# Pdf output
After serving the site, you can view the entire thesis at [here](http://127.0.0.1:8000/print_page/), which is printable.
